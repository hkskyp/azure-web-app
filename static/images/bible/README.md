# Bible Images Directory

## Placeholder Image

Create a `placeholder.webp` file in this directory with:
- Size: 800x300px
- Format: WebP
- Content: Soft gradient background (e.g., purple to blue)
- File size: < 50KB

## Verse Images

Generate images using Stable Diffusion for each verse:

1. Use prompts based on verse content
2. Convert to WebP format
3. Optimize to < 100KB
4. Name according to `bible_verses.py` (e.g., `psalm_23_1.webp`)

### Example Prompts

**Psalm 23:1** (`psalm_23_1.webp`):
```
A peaceful shepherd guiding sheep through green pastures,
golden sunlight, biblical art style, serene atmosphere
```

**Psalm 23:4** (`psalm_23_4.webp`):
```
A narrow valley path with dramatic lighting breaking through dark clouds,
hope and divine presence, biblical illustration style
```

**John 3:16** (`john_3_16.webp`):
```
Divine light descending from heaven to earth,
warm golden rays, spiritual love symbolism, renaissance style
```
